#   模板字符串
+   多行文本字符串
    ```
        - ES6的新语法中,可以使用反引号 ` 代替引号, 且字符串可以多行书写,例:
            var str =  `
            <div id="app">
                <p></p>
            </div>`;
    ```
+   字符串中的插值表达式
    ```
        - ES6的新语法中使用${}代替了+号拼接表达式,需要结合反引号使用,例:    
            var str = '123'
    
            var str = `张三${str}`;
    ```
