+   什么是宽放大模式
    ```
        - 如果一个模块很大,要分成几个部分,或者是一个模块要继承另一个模块就要使用宽放大模式;

        - 在浏览器环境中,模块的各个部分通常都是从网上获取的,有时无法知道哪个部分会先加载,这时候为了避免第一个执行的部分可能会加载一个不存在的空对象,这时就需要采用宽放大模式;

    ```
+   宽放大模式格式
    ```
        ;(function(add){
            add = function(a,b){
                return parseFloat(a) + parseFloat(b);
            }
            window.add = add;
        })(window.add||{})
    ```
