+   模块
    ```
        模块的概念类似于java中的package,逻辑上相关的代码组织到同一个包中,包内是一个相对独立的王国,也不用担心命名冲突,如果外部使用直接使用import导入即可;

        JavaScript在设计时,给js的定位原因,没有提供类似的功能,开发者需要模拟出类似的功能,来隔离,组织复杂的JavaScript代码,我们成为模块化;

        模块就是一个实现特定功能的文件,有了模块,我们就可以更方便的使用别人的代码,想要什么功能,就加载什么模块。模块开发需要遵循一定的规范;
        
        - 立即执行函数封装对象就是模块化的体现;
    ```
