# 包装类

+   在js 中提供了三个包装类可以把基本数据类型转为对象处理(但在开发中不会使用,比较起来比较麻烦)
+   new Number(3);

+   两个对象比较时比较的是虚地址

+   如果将对象和基本数据类型比较会隐式转换


+   基本数据类型为什么能调用构造函数的方法?
    ```
        - 基本类型的原始值在被当做对象使用的时候,JS将把他们包装成对象,使用完毕后,把它变为原始类型,并销毁其对象的性质;
    ```
+   字符串字面量能用点(.)调用构造函数的方法,为什么Number字面量不行?
    ```
        - 由于数字后的点(.)可能可能会是小数点,因此使用Number字面量调用构造函数中的属性要通过(1).toString() 这种形式;
    ```
