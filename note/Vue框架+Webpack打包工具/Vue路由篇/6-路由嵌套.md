#   路由嵌套
> 路由之间也可以有嵌套关系,注意下图中 foo 为参数
```
/user/foo/profile                     /user/foo/posts
+------------------+                  +-----------------+
| User             |                  | User            |
| +--------------+ |                  | +-------------+ |
| | Profile      | |  +------------>  | | Posts       | |
| |              | |                  | |             | |
| +--------------+ |                  | +-------------+ |
+------------------+                  +-----------------+
```
+   children
> chilren 用来设置子路由的匹配规则,例:
```
routes:[
    {path:'/',redirect:'/user'},
        {
            path:'/user',
            component:user,
            children:[
               {path:'profile',component:profile},
                {path:'posts',component:posts}
            ]
    }
]
```
