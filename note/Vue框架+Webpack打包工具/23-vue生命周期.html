+ 什么是声明周期 ``` 生命周期: 从Vue实例创建,运行,到销毁期间;伴随着各式各样的事件,这些事件统称为生命周期; 生命周期钩子: 指的是生命周期函数的别名 生命周期钩子 == 生命周期函数 == 生命周期事件 生命周期函数分类: 1. 创建期间的生命周期函数 beforeCreate :表示刚初始化了一个空的实例对象,这个时候实例对象身上只有一些默认的事件和生命周期函数,其他的属性和方法都为创建; (在beforeCreate生命周期函数执行的时候,data和methods中的数据都还没有初始化) created :这个时候data和methods中的数据已经初始化完毕,如果要使用data和methods中的数据最早只能在create中操作; beforeMount :这里表示已经把#app模板 编译完成了但是尚未渲染到页面中;此时输出dom节点中的数据仍然为插值表达式 mounted: 这里表示已经把内存中编译好的模板挂载到了页面中去,用户已经可以看到渲染好的页面了; 这时打印dom中的插值表达式已经被替换为data或者methods中的数据,此时如果没有别的操作的话该实例就静静的在内存中发呆; 2. 运行期间的生命周期函数 - 运行期的两个事件,当data改变的时候会被调用,有选择性的触发0次到n次 beforeUpdate: 该事件执行的时候,data中的数据已经改变,但是尚未渲染到页面中的dom元素中; update: 该事件执行的时候,data中的数据已经改变且已经同步到页面中去了; 3. 销毁期间的声明周期函数 beforeDestroy: 当调用vm.$destroy()的时候,该实例开始被销毁beforeDestroy函数被调用,但是此时vue实例身上基本所有的属性都仍然可用; destroyed: 当对象中的数据已经被销毁的时候调用destroy,结束该实例对象的生命; ```
