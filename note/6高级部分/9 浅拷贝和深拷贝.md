+   浅拷贝
    ```
        - 浅拷贝是指如果拷贝的对象中有对象,对象公用的是同一个对象;

        代码: function copy(o1,o2){
                for( var key in o1 ){
                    o2[key] = o1[key]; // 如果是对象,公用同一个对象
                }        
            }
    ```
+   深拷贝
    ```
        - 深拷贝是指如果拷贝对象的属性中如果有对象,就新建一个对象,把对象也拷贝一下,并不是共用一个对象

        代码: 
            function deepCopy(o1,o2){
                for( var key in o1){
                    var item = o1[key];
                    // 如果是对象,就拷贝一下该对象
                    if(item instanceof Object){
                        // 如果属性是一个对象
                        var o = {};
                        deepCopy(item,o);
                    }else if(item,arr){ 
                        // 如果属性是数组
                        var arr = [];
                        deepCopy(item,arr)
                    }else{
                        // 如果是普通数据类型的属性
                        o2[key] = item;

                    }
                }
            }
    
    ```
