+   属性的读写规则
    ```
        1. 读取对象的属性,如果没有则会按照原型链的方式查找,直到找到object的原型对象,object的原型为null

        2. 在添加一个属性的时候,会直接在对象中添加不会去原型中寻找有没有该属性;
    ```
+   注意事项1:
    ```
        1. 我们向对象中添加方法的时候,为了节省内存一般是在原型对象中添加;
            例: Student.prototype.sayHi(){ ... }

        2. 当一个对象要多个方法的时候,一个一个添加就比较麻烦,因此我们可以给构造函数中的prototype 重置赋值的形式添加多个方法

            例: Student.prototype = {
                    // 一般构造函数的原型对象中只有两个方法:
                    // 1. 字面量对象中的__proto__指向object中的原型,所以不需要添加__proto__属性
                    // 2. 添加构造函数属性:
                    constructor: Student,

                    // 新增方法
                    sayHi:function(){},
                    ...
            }
    ```
+   注意事项2:
    ```
        1. 给原型对象中添加方法, 在添加之前调用该对象的方法会报错;因为在原型对象添加方法之前是没有该方法的;
    ```
